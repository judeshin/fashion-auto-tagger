# Examples

This folder is for sample images to test the auto-tagger.

## Usage

1. Add your fashion product images here (PNG, JPG, etc.)
2. Run the demo:
   ```bash
   python demo.py
   ```
3. Enter the path to your image when prompted

## Recommended Test Images

- Hoodies
- Jackets
- Shirts
- Pants/Jeans
- Dresses

You can use images from:
- CLO 3D renders
- Product photography
- Fashion e-commerce sites

---

**Note:** This folder is included in `.gitignore` by default. If you want to commit example images to the repository, uncomment the relevant lines in `.gitignore`.
